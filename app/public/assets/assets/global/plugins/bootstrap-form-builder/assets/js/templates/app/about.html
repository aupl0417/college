About

Inspired By Adam Moore (@minikomi) to help take the stress out of writing all that markup to get bootstrap forms together. The original version of this was a big jQuery spaghetti mess, so as of March 2013 it has been rewritten using backbone.js and takes advantage of underscode.js templates.

This is the BootStrap 3 version. Rewritter the previous version using in-house Class implementation, and Mustache template language. Did not pick backbone.js because I already know it and want to learn and implment MVC Javascript framework by my own. This is also another learning/hacking experience for me.

Adding snippets is quite simple now (due to the rewrite!) so please don't hesitate to open an issue!
